## 3. 项目概述

### 核心功能
- **自动化战斗**: 自动副本战斗助手、技能使用自动化
- **危险区域显示**: 实时显示副本中的危险区域
- **触发器系统**: 自定义触发条件和动作
- **事件记录**: 游戏事件记录和回放
- **远程控制**: 远程控制功能
- **黑名单系统**: 玩家黑名单管理

### 项目边界
```yaml
范围内:
  - FF14 游戏辅助功能开发
  - 副本数据管理 (M11S, M12S 等)
  - UI 界面开发
  - 游戏事件监听和处理
范围外:
  - 游戏客户端修改
  - 破坏游戏平衡的功能
```
