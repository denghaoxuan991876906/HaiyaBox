# Triggers

## 职责

触发器系统，提供灵活的条件判断和动作执行机制。

## 接口定义

### 触发条件
| 类名 | 说明 |
|------|------|
| 检测目标位置 | 检测目标位置是否满足条件 |

### 触发动作
| 类名 | 说明 |
|------|------|
| 启动bmr | 启动 BMR 功能 |
| 使用技能动作 | 执行指定技能 |
| 指定职能tp指定位置 | 指定职能 TP 到指定位置 |
| 指定职能使用技能 | 指定职能使用特定技能 |

## 行为规范

### 触发器执行
**条件**: 触发条件满足
**行为**: 执行对应的触发动作
**结果**: 完成预定义的操作

### 条件判断
**条件**: 游戏状态变化
**行为**: 评估所有触发条件
**结果**: 返回条件是否满足

## 依赖关系

```yaml
依赖: Plugin, Data
被依赖: Plugin
```
