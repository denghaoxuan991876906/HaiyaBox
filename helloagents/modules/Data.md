# Data

## 职责

副本数据管理，存储和管理各种副本的战斗数据和机制信息。

## 接口定义

### 副本数据
| 副本 | 说明 |
|------|------|
| M11S 流星雨 | M11S 副本流星雨机制数据 |
| M12S 本体 | M12S 副本本体四运数据 |

### 数据结构
| 类 | 说明 |
|------|------|
| 流星雨数据 | M11S 流星雨机制的数据定义 |
| 四运数据 | M12S 四运机制的数据定义 |

## 行为规范

### 数据加载
**条件**: 进入对应副本
**行为**: 加载副本相关的战斗数据
**结果**: 数据可用于自动化战斗和危险区域显示

### 数据更新
**条件**: 副本机制变更
**行为**: 更新数据结构
**结果**: 新的机制生效
